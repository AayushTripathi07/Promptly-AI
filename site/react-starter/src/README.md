# Source Folder (src)

This folder contains the core React components, styles, and logic for your application.

## Key Files:
- `App.jsx`: The main layout and components of your site.
- `main.jsx`: The entry point that connects React to your HTML.
- `App.css`: Styles specific to the App component.
- `index.css`: Global styles for the entire site.

## How to Edit:
When you want to change the looks or behavior of your site, you will mostly be working inside this folder!
