# Legacy Static Website

This project contains the original static website code (HTML/CSS/JS) before migrating to React.

## How to Run

Since this is a simple static site, you can run it in a few ways:

### 1. Direct Open
Double-click `index.html` in your file explorer to open it in your browser.

### 2. VS Code "Live Server" (Recommended)
If you have the **Live Server** extension installed in VS Code:
1. Right-click `index.html`.
2. Select **Open with Live Server**.
3. The site will automatically open and reload whenever you save changes!

### 3. Using Vite (Optional)
If you want to use the modern toolset, you can run:
```bash
npx vite site/legacy-site
```

## Files
- `index.html`: The structure of the page.
- `styles.css`: The styling and layout.
- `script.js`: The interactive logic.
